# Superstore Sales Analysis

## What I Did
- Loaded and cleaned the Superstore dataset (9994 rows, 21 columns)
- No missing values or duplicates found
- Converted Order Date to datetime format

## Key Findings
- Total Sales: $2,297,200.86
- Total Profit: $286,397.02
- Profit Margin: 12.47%
- Top Product: Canon imageCLASS 2200 Advanced Copier ($61,599)
- Top Region: West ($725,457)
- Top Category: Technology ($836,154)
- Sales peak every Q4 (November/December)
